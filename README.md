## HouseHunt

HouseHunt is a user-centric house rental application that simplifies the real estate process for renters, buyers, sellers, and agents. It provides detailed property listings, real-time market insights, and advanced search filters based on location, price, and amenities.

The platform enhances user experience through features like virtual tours and seamless connectivity between stakeholders. It also streamlines documentation, making property transactions faster and more efficient.

With a focus on transparency, accuracy, and 24/7 accessibility, HouseHunt serves as a reliable and convenient digital marketplace for real estate needs.

## Tech Stack

## Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- CORS for cross-origin requests

## Frontend
- React 
- Vite for build tooling
- Tailwind CSS for styling
- Framer Motion for animations
- React Router for navigation
